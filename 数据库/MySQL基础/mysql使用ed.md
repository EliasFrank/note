MySQL登录

```mysql
mysql -h主机名 -P端口号 -u用户名 -p密码
本机登录可以简写成：myslq -u用户名 -p密码
```

---

MySQL的常见命令

```
1.查看当前所有的数据库
show databases;
2.打开指定的库
use 库名;
3.查看当前库的所有表
show tables;
4.查看其它库的所有表
show tables from 库名;
5.创建表
create table 表名(
	列名 列类型,
	列名 列类型,
	...
);
6.查看表结构
desc 表名;
```

------

MySQL的语法规范

1. 不区分带小写，但建议关键字大写，表名，列名小写

2. 每条命令最好用分号结尾

3. 每条命令根据需要，可以进行缩进或换行

4. 注释：

   ​	单行注释：#注释文字

   ​					：-- 注释文字（-- ）两个-后面有一个空格

   ​	多行注释：/* 注释文字 */
