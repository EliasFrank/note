### 什么是jvm

Java Virtual Machine - java 程序的运行环境（java二进制字节码的运行环境）

好处：

* 一次编写，到处运行

* 自动内存管理，垃圾回收功能
* 数组下标越界检查
* 多态

比较：

jvm  jre  jdk
